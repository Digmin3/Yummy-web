Yummy privacy policy

English

Information stored by the application

The application allows configuring a series of food intolerances available to the user.

The images taken with the camera are temporarily stored for analysis with a image recognition and are later discarded.

When accessing the application, a menu with recipes is generated and stored on the device. The recipes marked as favorites are stored on the device for reading without an internet connection.

No sensitive data that can identify the user is stored.

What do we collect it for

This information is used for the operation of the application. Not used for other purposes or sold to third parties.

Who has access to this information

The information is not synchronized over the internet, only the user of the device has access to it information.

To generate a menu that excludes user intolerances or analyze images, the list of intolerances or the image are sent during the process to the Spoonacular API, without any user identifier.

You can check Spoonacular's privacy policy here: https://spoonacular.com/terms

Where the information is processed and stored

The information is not synchronized over the internet, they are saved locally on the device.

Recipe information is obtained from the Spoonacular service. The application is responsible for displaying the information received through their screens.

Connections to this service are made securely over the internet.

To generate a menu that excludes user intolerances or analyze images, the list of intolerances or the image are sent during the process to the Spoonacular API, without any user identifier.

You can check Spoonacular's privacy policy here: https://spoonacular.com/terms



Política de privacidad de Yummy

Español

Información que almacena la aplicación

La aplicación permite configurar una serie de intolerancias alimentarias a disposición del usuario.

Las imágenes tomadas con la cámara son almacenadas de forma temporal para su análisis con un sistema de reconocimiento de imágenes y posteriormente son desechadas.

Al acceder a la aplicación, se genera un menú con recetas que se almacena en el dispositivo. Las recetas marcadas como favoritos se almacenan en el dispositivo para la lectura sin conexión a internet.

No se almacena ningún dato sensible que pueda identificar al usuario.

Para qué la recopilamos

Esta información se utiliza para el funcionamiento de la aplicación. No se utiliza para otros fines ni se vende a terceros.

Quién tiene acceso a esta información

La información no se sincroniza por internet, solo el usuario del dispositivo tiene acceso a esta información.

Para generar un menú que excluya las intolerancias del usuario o analizar las imágenes, la lista de intolerancias o la imagen se envían durante el proceso a la API de Spoonacular, sin ningún tipo de identificador de usuario.

Puedes consultar la política de privacidad de Spoonacular (en inglés) aquí: https://spoonacular.com/terms

Dónde se procesa y almacena la información

La información no se sincroniza por internet, se guardan de forma local en el dispositivo.

La información de las recetas se obtiene del servicio Spoonacular. La aplicación es la encargada de mostrar la información recibida a través de sus pantallas.

Las conexiones a este servicio se realizan de forma segura por internet.

Para generar un menú que excluya las intolerancias del usuario o analizar las imágenes, la lista de intolerancias o la imagen se envían durante el proceso a la API de Spoonacular, sin ningún tipo de identificador de usuario.

Puedes consultar la política de privacidad de Spoonacular (en inglés) aquí: https://spoonacular.com/terms
